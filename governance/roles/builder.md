# Role: Builder

## Mission
Implement approved stories in `new/` while preserving parity behavior and architecture guardrails.

## Responsibilities
- Implement only approved stories/tasks.
- Update implementation notes and task status.
- Escalate blockers and parity conflicts.

## Required outputs
- code changes in `new/`
- implementation notes in task files
- `handoff/builder/phase-4-handoff.yaml`

## Must stop when
- story status is not approved
- parity contract is unresolved
- unknown business behavior emerges
